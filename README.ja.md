## TwitterXZ とは？

[TwitterXZ](https://twitterxz.com) は、Twitter/X の動画や GIF をスマートフォンやパソコンに直接保存できる **Twitter 動画ダウンローダー** です。ツイートのリンクを貼り付けるだけで、無料で高画質ダウンロードが可能です。

---

## 主な機能

- **Twitter 動画ダウンロード** - Twitter/X の公開動画をすべてダウンロード可能
- **Twitter GIF ダウンロード** - GIF アニメを MP4 形式で保存
- **複数の画質を選択** - 360p / 720p / 1080p から選択可能
- **ログイン不要** - Twitter/X アカウント不要、完全無料
- **スマホ対応** - iPhone・Android でも快適に使用可能
- **オンラインツール** - アプリやプラグイン不要、ブラウザだけで完結

---

## 使い方

1. Twitter/X を開き、ダウンロードしたい動画ツイートを見つける
2. 共有ボタンをタップしてツイートのリンクをコピー
3. **[https://twitterxz.com](https://twitterxz.com)** にアクセス
4. 入力欄にリンクを貼り付けて「ダウンロード」をクリック
5. 画質を選んでデバイスに保存

## よくある質問

**Q: TwitterXZ は無料ですか？**
A: はい、[TwitterXZ.com](https://twitterxz.com) は完全無料です。基本機能の利用に登録は不要です。

**Q: 非公開アカウントの動画はダウンロードできますか？**
A: いいえ、公開ツイートの動画のみ対応しています。

**Q: Twitter Space の録音はダウンロードできますか？**
A: 現在は非対応です。動画と GIF のダウンロードに特化しています。

**Q: ダウンロードした動画にウォーターマークは入りますか？**
A: 入りません。Twitter のオリジナル動画ファイルをそのまま保存できます。

**Q: スマホでの使い方は？**
A: スマホのブラウザで [twitterxz.com](https://twitterxz.com) を開き、ツイートのリンクを貼り付けるだけです。アプリ不要です。

**Q: 対応フォーマットは？**
A: MP4 動画形式と GIF（MP4 に変換）に対応しています。

**Q: ダウンロード速度はどのくらいですか？**
A: 動画の解析は通常 3 秒以内に完了します。ダウンロード速度はお使いのネット環境によります。
