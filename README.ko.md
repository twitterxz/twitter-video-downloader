## TwitterXZ란?

[TwitterXZ](https://twitterxz.com) 는 Twitter/X의 동영상과 GIF를 스마트폰이나 컴퓨터에 직접 저장할 수 있는 **트위터 동영상 다운로더**입니다. 트윗 링크를 붙여넣기만 하면 무료로 고화질 다운로드가 가능합니다.

---

## 주요 기능

- **트위터 동영상 다운로드** - Twitter/X의 모든 공개 동영상 다운로드 지원
- **트위터 GIF 다운로드** - GIF 애니메이션을 MP4 형식으로 저장
- **다양한 화질 선택** - 360p / 720p / 1080p 화질 선택 가능
- **로그인 불필요** - Twitter/X 계정 없이도 완전 무료 이용
- **모바일 최적화** - iPhone 및 Android에서 원활하게 사용 가능
- **온라인 도구** - 앱이나 플러그인 설치 없이 브라우저에서 바로 사용

---

## 사용 방법

1. Twitter/X를 열고 다운로드하고 싶은 동영상 트윗을 찾기
2. 공유 버튼을 눌러 트윗 링크 복사
3. **[https://twitterxz.com](https://twitterxz.com)** 접속
4. 입력창에 링크를 붙여넣고 다운로드 클릭
5. 원하는 화질을 선택하여 기기에 저장

## 자주 묻는 질문 (FAQ)

**Q: TwitterXZ는 무료인가요?**
A: 네, [TwitterXZ.com](https://twitterxz.com) 은 완전 무료입니다. 기본 기능은 회원가입 없이도 이용 가능합니다.

**Q: 비공개 계정의 동영상도 다운로드할 수 있나요?**
A: 아니요, 공개 트윗의 동영상만 지원합니다.

**Q: Twitter Space 녹음도 다운로드할 수 있나요?**
A: 현재는 지원하지 않습니다. 동영상과 GIF 다운로드에 집중하고 있습니다.

**Q: 다운로드한 동영상에 워터마크가 있나요?**
A: 없습니다. 트위터 원본 동영상 파일 그대로 저장됩니다.

**Q: 스마트폰에서는 어떻게 사용하나요?**
A: 스마트폰 브라우저에서 [twitterxz.com](https://twitterxz.com) 을 열고 트윗 링크를 붙여넣으면 됩니다. 앱 설치 불필요합니다.

**Q: 어떤 형식을 지원하나요?**
A: MP4 동영상 형식과 GIF（MP4로 변환）를 지원합니다.

**Q: 다운로드 속도는 어느 정도인가요?**
A: 동영상 분석은 보통 3초 이내에 완료됩니다. 다운로드 속도는 인터넷 환경에 따라 다릅니다.
